# Problema 0 - Caso Sintético Médio

**Descrição:** Caso de teste com 5 links em área metropolitana de Fortaleza, 24 períodos horários e 3 tipos de carport PV.

**Dimensões:**
- Links (L): 5
- Períodos (T): 24 (horas do dia)
- Tipos PV (K): 3

**Características:**
- **Link 0:** Centro principal (alta demanda)
- **Link 1:** Centro secundário (alta demanda)
- **Link 2:** Zona comercial (média demanda)
- **Link 3:** Zona industrial (média demanda)
- **Link 4:** Zona residencial (baixa demanda)

**Horizonte de planejamento:** 10 anos

**Data de criação:** 2025-01-15

**Fonte:** Dados sintéticos baseados na tese de Caio dos Santos (Unicamp, 2021)
